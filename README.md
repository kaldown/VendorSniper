# VendorSniper

[![PayPal](https://img.shields.io/badge/Donate-PayPal-00457C?logo=paypal&logoColor=white)](https://www.paypal.com/donate/?hosted_button_id=FG4KES3HNPLVG)

If you find this useful, consider [supporting development](https://www.paypal.com/donate/?hosted_button_id=FG4KES3HNPLVG).

Auto-buy limited-supply vendor items when they restock in World of Warcraft Classic.

## Features

- **Auto-buy on restock** - Automatically purchases limited-supply items when a vendor restocks
- **Configurable item list** - Choose which limited-supply items to watch for
- **Minimap button** - Quick access via LibDBIcon

## Installation

1. Download and extract to your `Interface/AddOns/` folder
2. Ensure the path is: `Interface/AddOns/VendorSniper/VendorSniper.toc`

## Supported Game Versions

- Classic Era
- Anniversary
- Hardcore
